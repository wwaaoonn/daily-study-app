# Requirements

## Goal
Enable users to study something every day with minimal friction.

## Core Idea
A question is sent via email every morning.

## User Flow

1. User receives an email at 5:00 AM
2. User opens the link
3. User answers a multiple choice question
4. The result and explanation are shown
5. User can challenge more questions

## MVP Scope

- Daily email question
- 4-choice answers
- Result page
- Challenge more button
- Progress dashboard

## Out of Scope (initial)

- Mobile app
- AI recommendation
- Payment
